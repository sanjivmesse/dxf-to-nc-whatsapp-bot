# DXF to NC WhatsApp Bot

Automatic DXF to G-code converter with WhatsApp integration for DSPA11 CNC machine.

## Features
- WhatsApp bot for file upload
- DXF to G-code conversion
- Support for multiple end mills (3mm, 4mm, 5mm, 6mm, 8mm, 10mm, 12mm, 28mm)
- Material presets (Wood, Acrylic)
- Max RPM: 12000
- Raspberry Pi Zero 2W optimized

## Hardware
- Raspberry Pi Zero 2W
- 16GB SD Card
- WiFi connection
- DSPA11 CNC Machine

## Installation
See `INSTALL.md` for detailed setup instructions.

## Usage
1. Send DXF file to WhatsApp bot
2. Select end mill size
3. Choose material
4. Bot generates G-code
5. Load in DSPA11
